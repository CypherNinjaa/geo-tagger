How it works:

  1. Face Detection: Detects faces using SSD MobileNet v1
  2. Landmark Detection: Identifies 68 facial landmarks including 12 eye points
  3. EAR Calculation: Eye Aspect Ratio = (|p2-p6| + |p3-p5|) / (2 * |p1-p4|)
  4. Blink Detection: When EAR drops below threshold for consecutive frames
  5. Liveness Check: Requires multiple blinks to prove user is alive
